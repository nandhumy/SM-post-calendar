# Task 1 – Social Media Post Calendar

## 📌 Objective

The objective of this task was to create a **30-day social media content calendar** for a brand, including captions, hashtags, and visual content ideas to improve audience engagement and brand visibility.

---

## 🎯 Task Description

A structured social media posting plan was developed to ensure consistent brand communication and audience interaction across the selected platform.

The calendar includes:

* Daily post ideas
* Marketing-focused captions
* Relevant hashtags
* Image/design suggestions

---

## 🛠 Work Done

* Planned a 30-day posting strategy
* Designed engaging and audience-focused captions
* Selected trending and niche hashtags
* Created visual content concepts for each post
* Organized the calendar in Excel format

---

## 📂 Deliverable

* `social_media_calendar.xlsx` – Complete 30-day social media content calendar

---

* Social Media Marketing
* Content Planning
* Brand Communication
* Audience Engagement Strategy
